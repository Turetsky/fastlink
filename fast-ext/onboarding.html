Stay in the loop

Watch Claude work

FastLink pins the tab Claude is driving (its target tab), so Claude keeps driving that tab even when you switch away — you never lose your place in the chat. You've got several ways to see exactly what's happening:

  1. On the driven tab — a “Claude is driving this tab” panel (top-right) annotates each click, type, and read in real time.
  2. The FastLink side panel — open it from Chrome's side-panel button to mirror Claude's live transcript and current action. It stays visible across all tabs, so it's the best way to watch from anywhere.
  3. Floating transcript — a compact transcript overlay can also follow you onto whatever tab you're currently viewing.
  4. Toolbar badge — at a glance: a spinner means working, “!” means possibly stuck, and empty means idle. The popup shows which tab is being driven plus the last action.

Tip: keeping claude.ai in one window and the driven tab in another side-by-side still works nicely — but with the side panel it's now optional, since you can watch from any tab.

Reopen this page anytime from the extension's options. The local broker (Claude Code / Desktop) keeps working alongside the cloud relay.
