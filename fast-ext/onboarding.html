Welcome to FastLink

FastLink lets Claude drive your own Chrome tab — read the page, click, fill forms, capture console & network. Two quick steps to connect.

Enable vision optional

FastLink works fully without this — it reads and drives the page from the DOM. Add your own Google Gemini API key only to light up the optional vision / scout speed tier (screenshot-based locating and page pre-reads). Without a key, those specific tools return {disabled:true} and everything else works normally.

3

Add FastLink to claude.ai

Add this URL as a custom connector in claude.ai, then sign in with the same account. Claude can then drive this browser.

https://relay.ytx.app/mcp
  1. In claude.ai, open Settings → Connectors → Add custom connector.
  2. Paste the URL above and save, then Connect and sign in with the same account.
  3. Start a chat and ask Claude to use FastLink on your active tab.
4

Watch Claude work

FastLink pins the tab Claude is working on, so it keeps driving that tab even when it isn't focused — you never lose your place in the chat.

  1. Keep claude.ai in one window and put the tab you want driven in a second window, side-by-side.
  2. Ask Claude to act on that tab. A “Claude is driving this tab” panel appears top-right on it and annotates each click, type, and read in real time.
  3. Because the tab is pinned, Claude drives it in the background without stealing your focus. The extension toolbar icon and popup also show what's being driven.

You can reopen this page anytime from the extension's options. The local broker (Claude Code / Desktop) keeps working alongside the cloud relay.
