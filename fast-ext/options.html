Cloud relay control

The local broker (Claude Code / Desktop) is always on by default. Use this to turn the cloud relay OFF and run the local broker only. Does not revoke the device on the relay server — re-enable by pairing again.

Advanced control …

FastLink works out of the box with DOM-based reading, clicking, and form-filling. Advanced control grants Chrome's debugger permission, which adds: precise coordinate clicks/typing (vision tier), running scripts on the page, and capturing tabs that aren’t in front (so Claude can drive a background tab while you stay on the chat). Grant it only if you want those — you can revoke it anytime.
